### 前言

说是前言，其实是我的一些碎碎念。在当下这个AI十分发达的时代，写代码已经不再成为一件难事了，说实话，我自己也经常在用各种AI工具帮助我写代码。就个人来说，这种AI Coding的方式确实大大缩短了我做各种项目所需要的时间。但时间一长，我发现我变得**一行**代码都写不出来，对于问题独立思考的能力也在逐渐下降，更多的我会下意识的询问AI。我并不反对使用AI解答，但是我认为AI更多的应该是工具，而不是代替思考的大脑，尤其在当下这个注重解决问题能力的时代，我们生而为人的这种独立思考能力应该更加重要。<br>
另外呢，我发现了，即使AI如此的发达，许多编程工具也难以给出一个很棒的学习规划。我前几天还在尝试过用Claude Code给我编写PyTorch的学习计划，但是给出的也仅仅是一个很粗略的，很难说称之为“教程”的东西。而网上已有的博客教程又可能有一些专业化（？所以我也想要来尝试创建一份称之为“入门指南”的东西吧。

### 你能学到什么

通过这个系列学习，你将能够在我的带领下轻轻叩开编程世界的门扉吧。（ps:博主也不是什么编程大佬，只能说给各位充当一个引路人的身份吧
我将以我自己的理解，给各位介绍Python的基础语法、函数、面向对象等一系列概念。最后呢，我也给大家提供了如下几个Python案例：

- Python Django 用 Python 写一个自己的网站
- Python Mathplotlib 用 Python 绘制精美科研图片
- Pytorch 用 Python 编写一个自己的人工智能模型

当然，因为博主的能力有限，文中的部分难免有讲解不充分的地方，欢迎大家在评论区讨论和分享自己的观点。

### 如何学

下面是我对于Python的一些学习方法的分享吧，我也算是断断续续接触了6年的代码吧。其实重点只有一句话。那就是**多写**。<br>
代码说白了就像是一门语言，写多了呢，你就自然而然产生了那种类似**语感**的东西。所以，在这儿鼓励大家多多练习coding，不要害怕报错。
